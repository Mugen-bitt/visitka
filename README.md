# Визитка — Pavel Evstigneev

Персональная визитка QA Automation Engineer в стиле Ghost in the Shell (2017).

## Особенности

- Светлый минималистичный дизайн в стиле GITS 2017
- Динамичный глитч-эффект с горизонтальными полосами
- Мультиязычность: RU, EN, JP, CN
- Эффект скремблирования текста при смене языка
- Адаптивный дизайн для мобильных устройств

## Архитектура

```
visitka/
├── index.html      — Главная страница с анимациями и переводами
├── style.css       — Стили: GITS 2017, глитч-эффекты, адаптив
├── index-gits.html — Альтернативная версия (тёмная тема GITS)
├── style-gits.css  — Стили для тёмной темы
├── service.js      — Express-сервер (опционально)
└── 404.html        — Кастомная 404 страница
```

## Технологии

| Компонент | Описание |
|-----------|----------|
| Шрифт | Share Tech Mono |
| Цветовая схема | Светлый фон, сине-фиолетовые акценты |
| Анимации | CSS + JavaScript глитч-эффекты |
| Скремблирование | Посимвольная анимация с поддержкой CJK |

## Запуск

**Локально:**
```bash
# Просто открыть index.html в браузере
# или
python3 -m http.server 8080
```

**С Express-сервером:**
```bash
npm install express node-fetch cors
node service.js
# http://localhost:3000
```
